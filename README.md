# LearnPath — AI-Powered Personalized Learning, Assessment, and Career System

LearnPath is a Flask-based web application that generates personalized learning paths using AI, tracks daily progress, provides tests, manages study notes, supports reminders, and includes career tools like resume building and PDF generation.

## Features

- AI-generated personalized learning paths
- Daily learning schedule by topic and duration
- YouTube video suggestions for each day
- Day-wise progress tracking
- Learning streak tracking
- Test system with multiple attempts
- Profile page with activity chart
- Study notes saving and PDF export
- Resume builder
- To-do management
- PDF summarizer
- Certificate generation on completion
- Parent dashboard and notifications
- Email reminders for learning and overdue tasks

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask
- Database: SQLite
- AI API: OpenRouter
- Video API: YouTube Data API
- PDF Generation: ReportLab
- Charts: Chart.js
